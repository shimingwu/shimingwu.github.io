## Consistency Levels

- Strong Consistency
  - See all previous writes
- Eventual Consistency
  - See subset of previous writes
- Consistent Prefix
  - See initial sequence of writes
- Bounded Staleness
  - See all "old" writes
- Monotonic Reads
  - See increasing subset of writes
- Read My Writes
  - See all writes performed by reader
